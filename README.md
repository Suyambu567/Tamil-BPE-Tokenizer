# Tamil BPE Tokenizer From Scratch

A custom Tamil Byte Pair Encoding (BPE) tokenizer implemented from scratch using Python.

## Features

- Tamil Unicode support
- Character tokenizer
- Corpus loader
- Frequency counter
- BPE training
- Merge rules learning
- Encode / Decode
- Save model
- Load model

## Project Structure

TAMIL_TOKENIZER/
├── data/
├── models/
├── scripts/
└── src/

## Example

Input:

நான் பள்ளிக்கு

Output:

['நான்', 'பள்ளிக்கு']
